# Hacking

Eclectic mix of stand-alone python scripts.

## Installation

1. Create a virtual env:
   python -m venv .venv
   source .venv/bin/activate   # or `.venv\Scripts\activate` on Windows

2. Install requirements (if any):
   pip install -r requirements.txt
